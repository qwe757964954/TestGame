## 📐 命名规范

**格式：** `ui_模块_功能_状态.png`

| 字段 | 说明 | 示例 |
|---|---|---|
| ui_ | 统一前缀，便于搜索筛选 | `ui_` |
| 模块 | 所属页面/模块 | `home`（首页）、`game`（游戏内）、`shop`（商店） |
| 功能 | 资源功能类型 | `icon`（图标）、`btn`（按钮）、`bg`（背景）、`bar`（栏） |
| 状态 | 交互状态（可选） | `normal`（默认）、`pressed`（按下）、`disabled`（禁用） |

**示例：**
- `ui_home_icon_coin.png` - 首页金币图标
- `ui_home_btn_start_normal.png` - 首页开始按钮（默认状态）
- `ui_home_logo_title.png` - 首页游戏标题Logo

## 🎨 设计规范

### 视觉风格
- **材质表现：** PBR物理渲染材质，区分金属、玻璃、塑料、硅胶、果冻等不同质感
- **光影系统：** 统一自然光方向（左上45°），所有元素高光、投影保持一致，有体积感
- **圆角规范：** 所有按钮、面板、图标统一大圆角，呈现果冻Q弹饱满质感
- **色彩体系：** 马卡龙色系为主，饱和度适中，蓝色果冻底座统一识别
- **图标结构：** 功能图标统一采用「蓝色果冻底座 + 3D立体元素」的组合结构
- **角色设计：** 吉祥物采用圆润球体造型，软胶/硅胶质感，次表面散射效果
- **场景氛围：** 背景采用花园草地场景，层次丰富，有自然光影和氛围感

### 尺寸使用建议
- 开发时可根据实际需求缩放，建议保持2的幂次尺寸
- 高清屏建议使用 @2x 或 @3x 倍率
- 图标建议最小使用尺寸不低于 64×64 px（保证细节清晰）

## 🚀 使用说明

1. **图集打包：** 按 atlas_ 前缀的目录分组打包成Sprite Atlas
2. **资源加载：** 按模块按需加载，首页只加载 home 相关图集
3. **命名保持：** 建议代码中资源名与文件名保持一致，便于维护
4. **扩展规则：** 新增资源遵循相同命名规范和目录结构
